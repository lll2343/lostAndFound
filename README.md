> 
> author: lyz && dlf

## Build Setup

开发

``` bash
# 代码下载
git clone https://github.com/lll2343/lostAndFound.git

# 进入项目目录
cd fundandlost

# 安装依赖
npm install

# 开发时构建
npm run dev
```

发布

```bash
# 打包构建
npm build

# 指定平台的开发时构建(微信、百度、头条、支付宝)
npm dev:wx
npm dev:swan
npm dev:tt
npm dev:my

# 指定平台的打包构建
npm build:wx
npm build:swan
npm build:tt
npm build:my

# 生成 bundle 分析报告
npm run build --report
```
